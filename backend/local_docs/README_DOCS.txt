PLEASE READ:

This directory `local_docs` is where you must place your Civic Information documents.

1. Find PDF documents relevant to your village/community (e.g., "Maternal_Health_Schemes.pdf", "Crop_Insurance_Guidelines.pdf").
2. Paste them into this folder.
3. Run the ingestion script:
   python ../ingest_local.py

The script will read all *.pdf files here and train the local AI brain.
