# Relevant Open Datasets for CivicBridge

The following public datasets directly power CivicBridge features: issue heatmaps, service directories, budget transparency, and demographic context.

| Dataset | Source / URL | CivicBridge Use |
| :--- | :--- | :--- |
| **311 Service Requests** | data.cityofchicago.org / NYC Open Data | Issue tracking & heatmap |
| **US Census (ACS)** | census.gov/data/developers | Ward/district demographics |
| **Gov Budget Data** | usaspending.gov API | Budget transparency module |
| **Public Infrastructure** | data.gov – roads, bridges | Infrastructure issue linking |
| **Crime Reports** | FBI NIBRS / local open data | Safety & incident overlay |
| **Air Quality Index** | openaq.org / EPA AQS API | Environment widget |
| **Public Transit GTFS** | transitfeeds.com / agency GTFS | Transport service finder |
| **Voting Districts** | census.gov TIGER/Line | District-based routing |
| **Flood / Hazard Zones** | FEMA NFHL | Hazard alert feature |
| **Govt Contacts** | USA.gov API / local CSV | Service directory |
| **Parks & Recreation** | data.gov parks dataset | Local amenity map |
| **School Zones** | NCES / state open data | Community info panel |
